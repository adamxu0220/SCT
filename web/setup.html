For the first time login, please setup your PC environment.Click the hyperlinks to install

the programs and start client-side.

1.  Install python-2.7.1

2.  Install pyserial-2.5

3.  Install PIL-1.1.7

4.  Install pywin32

5.  Copy and unzip test client folder to your PC

6.  Add python executable file path(e.g. -- C:\Python27) to PC's environment path

7.  Run command on client PC, e.g. --

    E:\case\test_client>python server_client_new.py [10.37.130.212 or 10.37.142.233] [serial_port]
